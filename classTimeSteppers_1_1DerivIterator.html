Detailed Description

template<typename DerivVars>
class TimeSteppers::DerivIterator< DerivVars >

Iterator over the derivatives stored in a History object.

This returns typed data, and so cannot be obtained from an UntypedHistory. Type-erased derivatives can be obtained from HistoryIterator::derivative().
