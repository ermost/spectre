       5             : ## Installing SpECTRE on Apple Silicon {#installation_on_apple_silicon}
       6             : 
       7             : The following instructions show how to install SpECTRE and its dependencies
       8             : on Apple Silicon Macs. Apple Silicon is an arm64 architecture, in contrast
       9             : to the x86-64 architecture that SpECTRE usually targets. These instructions
      10             : will result in an Apple Silicon native build of SpECTRE.
      11             : 
      12             : \note Floating-point exception trapping is not currently
      13             : supported on Apple Silicon.
      14             : 
      15             : ### 0. Install the xcode command-line tools.
      16             : 
      17             : Install the xcode command-line tools, which include the clang compiler, etc.
      18             : 
      19             : ```
      20             : xcode-select --install
      21             : ```
      22             : 
      23             : ### 1. Make a directory to install prerequisites
      24             : 
      25             : First, make a directory to hold some prerequisites that spectre depends on.
      26             : Name this directory whatever you like, and set `SPECTRE_DEPS_ROOT` to its value.
      27             : These instructions, as an example, set this to the `apps` directory in the
      28             : user's home folder.
      29             : ```
      30             : export SPECTRE_DEPS_ROOT=$HOME/apps
      31             : mkdir $SPECTRE_DEPS_ROOT
      32             : cd $SPECTRE_DEPS_ROOT
      33             : mkdir src
      34             : cd src
      35             : ```
      36             : 
      37             : ### 2. Install python dependencies
      38             : 
      39             : Spectre depends on python and some python packages. There are different ways to
      40             : install an arm64-native python stack. The following instructions show how
      41             : to do this using [Miniforge](https://github.com/conda-forge/miniforge),
      42             : which supports Apple Silicon natively.
      43             : 
      44             : #### Miniforge
      45             : Distribution Miniforge is a way to install an arm64-native python stack on Apple
      46             : Silicon Macs. Download the
      47             : [Miniforge3-MacOSX-arm64.sh](https://github.com/conda-forge/miniforge/releases/latest/download/Miniforge3-MacOSX-arm64.sh)
      48             : installation script, and then do the following:
      49             : ```
      50             : # Install miniforge. Accept all default choices
      51             : bash Miniforge3-MacOSX-arm64.sh
      52             : # Activate conda base environment for current session
      53             : eval "$(${HOME}/miniforge3/bin/conda shell.zsh hook)"
      54             : # Activate conda at startup (to make sure you're always using native python)
      55             : conda init zsh
      56             : exit # close shell. conda will automatically load when new zsh shells start.
      57             : ```
      58             : 
      59             : Open a new terminal window. Then, install the necessary python packages.
      60             : ```
      61             : conda install --file support/Python/requirements.txt \
      62             :   --file support/Python/dev_requirements.txt
      63             : ```
      64             : 
      65             : You might also wish to install jupyter notebook support, using
      66             : ```
      67             : conda install -c conda-forge jupyterlab
      68             : ```
      69             : 
      70             : ### 3. Install dependencies with Homebrew
      71             : 
      72             : Most of spectre's dependencies beyond python can be installed using the
      73             : [homebrew](https://brew.sh) package manager. First, if you haven't
      74             : already, install Homebrew by
      75             : following the instructions on the [homebrew](https://brew.sh) homepage. Then,
      76             : run the following to install a fortran compiler and other dependencies:
      77             : ```
      78             : brew install gcc
      79             : brew install openblas boost gsl cmake doxygen
      80             : brew install ccache autoconf automake jemalloc hdf5 pybind11 yaml-cpp
      81             : ```
      82             : 
      83             : ### 4. Install remaining dependencies
      84             : 
      85             : Here, install the remaining dependencies that cannot be installed
      86             : with homebrew or miniforge.
      87             : 
      88             : ```
      89             : export SPECTRE_DEPS_ROOT=$HOME/apps
      90             : ```
      91             : 
      92             : ```
      93             : cd $SPECTRE_DEPS_ROOT
      94             : mkdir blaze
      95             : pushd blaze
      96             : curl -L https://bitbucket.org/blaze-lib/blaze/downloads/blaze-3.8.tar.gz \
      97             : > blaze-3.8.tar.gz
      98             : tar -xf blaze-3.8.tar.gz
      99             : mv blaze-3.8 include
     100             : popd
     101             : 
     102             : git clone https://github.com/edouarda/brigand.git
     103             : 
     104             : # Need master branch of libxsmm to support Apple Silicon
     105             : git clone https://github.com/hfp/libxsmm.git
     106             : pushd libxsmm
     107             : make
     108             : popd
     109             : 
     110             : pushd ./src
     111             : git clone https://github.com/Libsharp/libsharp.git
     112             : cd Libsharp
     113             : 
     114             : # Do not use compiler flag -march=native (unsupported on Apple Silicon)
     115             : sed "s/-march=native//" configure.ac > configure.ac.mod
     116             : mv configure.ac.mod configure.ac
     117             : 
     118             : autoupdate
     119             : autoconf
     120             : ./configure
     121             : make
     122             : mv auto $SPECTRE_DEPS_ROOT/libsharp
     123             : popd
     124             : ```
     125             : 
     126             : Next, install charm++. Note that Apple Silicon Macs require v7.0.0.
     127             : 
     128             : ```
     129             : git clone https://github.com/UIUC-PPL/charm
     130             : pushd charm
     131             : git checkout v7.0.0
     132             : ./build LIBS multicore-darwin-arm8 --with-production -g3 -j
     133             : popd
     134             : ```
     135             : Lastly, install Catch2. Note that catch2 v2.13.7 is required, and homebrew
     136             : can only install v3. Catch2 is a header only package, so no need to build
     137             : separately.
     138             : 
     139             : ```
     140             : git clone https://github.com/catchorg/Catch2.git
     141             : pushd Catch2
     142             : git checkout v2.13.7
     143             : popd
     144             : ```
     145             : 
     146             : ### 5. Clone, configure, and build SpECTRE.
     147             : Next, clone SpECTRE, make a build directory, and configure. In whatever
     148             : directory you prefer, clone SpECTRE and make a build directory, e.g.
     149             : ```
     150             : git clone git@github.com:sxs-collaboration/spectre.git
     151             : cd spectre
     152             : mkdir build
     153             : cd build
     154             : ```
     155             : 
     156             : Next, configure SpECTRE using the following `cmake command`.
     157             : 
     158             : ```
     159             : cmake -D CMAKE_C_COMPILER=clang -D CMAKE_CXX_COMPILER=clang++ -D \
     160             : CMAKE_Fortran_COMPILER=gfortran -D BUILD_PYTHON_BINDINGS=ON \
     161             : -D MEMORY_ALLOCATOR=SYSTEM \
     162             : -D CHARM_ROOT=${SPECTRE_DEPS_ROOT}/charm/multicore-darwin-arm8 \
     163             : -D BLAS_ROOT=$(brew --prefix openblas) -D CMAKE_BUILD_TYPE=Debug \
     164             : -D DEBUG_SYMBOLS=ON -D USE_PCH=ON \
     165             : -D SPECTRE_TEST_TIMEOUT_FACTOR=5 \
     166             : -D LIBXSMM_ROOT=${SPECTRE_DEPS_ROOT}/libxsmm/ \
     167             : -D BLAZE_ROOT=${SPECTRE_DEPS_ROOT}/blaze/ \
     168             : -D BRIGAND_ROOT=${SPECTRE_DEPS_ROOT}/brigand/ \
     169             : -D LIBSHARP_ROOT=${SPECTRE_DEPS_ROOT}/libsharp/ \
     170             : -D CATCH_INCLUDE_DIR=${SPECTRE_DEPS_ROOT}/Catch2/include/ \
     171             : -D Boost_ROOT=$(brew --prefix boost)/ \
     172             : -D CLANG_TIDY_BIN=$(brew --prefix llvm)/bin/clang-tidy \
     173             : -D BUILD_SHARED_LIBS=OFF \
     174             : -DGSL_ROOT=$(brew --prefix gsl)/include \
     175             : -DGSL_LIBRARY=$(brew --prefix gsl)/lib/libgsl.a ..
     176             : ```
     177             : 
     178             : Finally, build and test SpECTRE. E.g., on a Mac with 10 cores,
     179             : ```
     180             : make -j10
     181             : make -j10 test-executables
     182             : ctest --output-on-failure -j10
     183             : ```
